Google Sheet setup

Takes about 2 minutes. Do this once.

STEP 01

Create a new Google Sheet

Open Google Sheets and create a new spreadsheet. Name it "Legora Event Leads".

STEP 02

Open Apps Script

In the sheet, click Extensions → Apps Script. Delete any existing code and paste this:

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); // Add header row if sheet is empty if (sheet.getLastRow() === 0) { sheet.appendRow([ "Timestamp","Event","First Name","Last Name", "Email","Company","Job Title","# Lawyers", "Phone","Rating","Notes" ]); } sheet.appendRow([ data.timestamp, data.event, data.firstName, data.lastName, data.email, data.company, data.jobTitle, data.lawyers, data.phone, data.rating, data.notes ]); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); }
STEP 03

Deploy as web app

Click Deploy → New deployment. Select type: Web app. Set "Who has access" to Anyone. Click Deploy and copy the URL.

STEP 04

Paste the URL back in the app

That's it. Paste the URL into the webhook field and start logging.
